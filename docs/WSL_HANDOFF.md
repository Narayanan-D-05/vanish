## 🆘 Troubleshooting

### If `npm run compile:circuits` fails in WSL:

**Check Node.js version:**
```bash
node --version  # Should be v18+
```

**Install circom2 globally if missing:**
```bash
npm install -g circom2
```

**Manual compilation (if build script fails):**
```bash
cd circuits
npx circom2 shield.circom --r1cs --wasm --sym -o . -l ../node_modules
npx snarkjs groth16 setup shield.r1cs powersOfTau28_hez_final_12.ptau shield_final.zkey
npx snarkjs zkey export verificationkey shield_final.zkey shield_verification_key.json
npx snarkjs zkey export solidityverifier shield_final.zkey ../contracts/shieldVerifier.sol
```
